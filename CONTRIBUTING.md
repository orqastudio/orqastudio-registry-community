# Contributing Community Plugins

**Coming soon** — the community plugin registry is being set up.

OrqaStudio is in active pre-release development. The plugin SDK and submission process are not yet stable.

## Current Status

We're not accepting community plugin submissions yet while we establish the plugin SDK, distribution infrastructure, and review process.

If you're interested in building plugins for OrqaStudio, you're welcome to explore the [monorepo](https://github.com/orqastudio/app) and the plugin templates within it.

## Get In Touch

Join the conversation at [OrqaStudio Discussions](https://github.com/orgs/orqastudio/discussions) — whether you want to contribute, have questions, or just want to follow along.

## What's Coming

- Stable plugin SDK with documented APIs
- Plugin scaffold templates (`orqa plugin create`)
- Automated submission and verification process
- Curated registry with compatibility checks
